TextSend - [textsend.live](https://textsend.live/)

TextSend was designed to make it easier to send small scrips or code snippets to students on their chromebooks.

Features:
  1. Instant sharing
  2. one-click recieve
  3. backed by firebase

Many classrooms struggle with sending infomation / code snippets to students easily.

How it works:
  1. Paste a script into the website
  2. a unique 5-character ID is generated
  3. The script is saved in firebase
  4. Type the unique ID and click load script to view snippet.

Tools used:
  1. HTML, CSS, and JavaScript
  2. Firebase Firestore
  3. GitHub Pages
